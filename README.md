# 基于 ESP32 的超声波测距与云端监控系统

> 构建了一套本地显示与云端远程监控协同的非接触测距方案。

## 项目简介

基于 ESP32 主控 + HC-SR04 超声波传感器实现非接触测距，通过 I2C 驱动 LCD1602 本地实时显示，利用 MQTT 协议将数据上传至云端，支持 PC 端和移动端远程查看。双平台（Wokwi 仿真 + 物理硬件）验证通过。

## 系统架构

```
HC-SR04 传感器 ──▶ ESP32 ──I2C──▶ LCD1602（本地显示）
                     │
                     ▼ MQTT
            云端服务器 ──▶ PC/移动端（远程监控）
```

## 关键技术点

### 1. 双阈值滞回滤波算法
- 针对超声波测距在临界距离处频繁跳变的问题
- 设计双阈值滞回机制，消除测量临界抖动
- 稳态测距精度 **±1cm**

### 2. MQTT 云端对接
- 利用 MQTT 协议实现设备到云端的数据传输
- 支持 PC 端与移动端远程实时查看测量数据
- 具备基础的 JSON 数据解析与封包能力

### 3. 仿真 + 实物双平台验证
- **Wokwi 在线仿真**：快速验证逻辑，无需等待硬件
- **物理硬件测试**：ESP32 + HC-SR04 真实环境测试
- 72 小时连续运行，系统稳定性 **98%**
